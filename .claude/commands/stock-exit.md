# stock-exit

基于 Moneyball 框架，给出止盈条件、止损触发逻辑和减仓节奏。
**支持三种股票类型：🏦 蓝筹长持 / 🚀 翻倍候选 / ⚡ 热赛道动量**，止损和减仓逻辑完全不同。

## Arguments

Required: `<TICKER>` — 股票代码（如 `POET`, `NVDA`）

## 框架依据

读取 `wiki/frameworks/moneyball.md` 了解退出逻辑。
读取 `wiki/frameworks/sepa.md` 了解 Stage 退出信号。

## Steps

### 1. 读取入场信息 + 股票类型

检查 `data/tickers/[TICKER]/entry-*.md`（最新一份）提取：
- 入场价、止损位、目标1、目标2、入场日期
- **股票类型**（🏦 蓝筹长持 / 🚀 翻倍候选 / ⚡ 热赛道动量）— 决定整个退出逻辑
- 是否已触发自由股规则（翻倍候选）
- 移动止损当前值（动量）

如无入场文件，让用户提供：入场价、止损价、股票类型、当前持仓比例。

### 2. 拉取当前价格数据 + ATR（yfinance）

```python
import yfinance as yf
import pandas as pd

t = yf.Ticker("TICKER")
hist = t.history(period="1y")

# ATR-14
high_low   = hist['High'] - hist['Low']
high_close = abs(hist['High'] - hist['Close'].shift())
low_close  = abs(hist['Low']  - hist['Close'].shift())
tr = pd.concat([high_low, high_close, low_close], axis=1).max(axis=1)
atr14 = tr.rolling(14).mean().iloc[-1]

price  = hist['Close'].iloc[-1]
ma50   = hist['Close'].rolling(50).mean().iloc[-1]
ma150  = hist['Close'].rolling(150).mean().iloc[-1]
ma200  = hist['Close'].rolling(200).mean().iloc[-1]

# 周收盘（蓝筹 MA 止损判断）
weekly = hist['Close'].resample('W').last()
ma10w  = weekly.rolling(10).mean().iloc[-1]

# 近10日最高（动量移动止损）
hi10d = hist['High'].rolling(10).max().iloc[-1]
trailing_stop = hi10d - 1.5 * atr14

vol_today  = hist['Volume'].iloc[-1]
vol_avg20  = hist['Volume'].rolling(20).mean().iloc[-1]
vol_ratio  = vol_today / vol_avg20

print(f"现价={price:.2f}  ATR14={atr14:.2f}({atr14/price*100:.1f}%)")
print(f"MA50={ma50:.2f}  MA150={ma150:.2f}  MA200={ma200:.2f}")
print(f"10周线={ma10w:.2f}  成交量比={vol_ratio:.1f}x")
print(f"近10日高={hi10d:.2f}  移动止损参考={trailing_stop:.2f}")
```

### 3. 止损规则（三类型，按类型选择）

#### 🏦 蓝筹长持止损规则

**核心原则：不用机械价格止损，用论文止损。**
蓝筹股护城河深、短期波动大，固定百分比止损会过早被洗出。

**出仓条件（以下任一触发）：**
1. **论文破坏**（最优先）：护城河受损、竞争格局恶化、管理层重大失误、盈利能力持续下滑
2. **估值严重高估**：P/FCF 超过历史均值 50% 且无增长加速支撑
3. **LTCG 意识**：若持有未满 1 年，避免非必要出仓（税务成本 15-22%）
4. **Stage 4 + 基本面恶化同时发生**：两者缺一不确认出仓

> **不触发**：价格下跌 10-15% 本身不是出仓信号（蓝筹日常回调）。只有基本面恶化才出。

#### 🚀 翻倍候选止损规则

双重止损，谁先触发谁生效：

**① 价格止损（噪音过滤版）**
- 连续 **2个交易日收盘** 低于止损位才确认触发（单日跌破不算）
- 止损位 = 入场价 - max(2×ATR, 10%)，上限 20%

**② 论文止损（优先级更高）**
- 核心催化剂消失（合同取消、竞争对手突破、管理层造假）→ 立即出
- 融资稀释严重（>20% 股份增发，且无对应业务价值）→ 立即出
- 现金跑道 <3 个月 → 评估是否出

**自由股特殊规则（+100% 后）：**
- 已卖出 50% 后，剩余 50% 零成本持有
- 零成本持有后：不设价格止损（成本为零），目标 3-5×
- 唯一出仓条件：论文彻底破坏（公司基本面不再支撑翻倍逻辑）

#### ⚡ 热赛道动量止损规则

**核心：移动止损 + 赛道止损，两者谁先触发谁生效**

**① 移动止损（每日更新）**
```
移动止损 = 近10日最高价 - 1.5×ATR14
价格创新高 → 止损上移
价格收盘低于移动止损 → 出仓
```

**② 赛道止损（赛道 ETF 信号）**
- 赛道 ETF 收盘跌破 MA50 → 不管个股价格，立即评估出仓
- 赛道 ETF 连续 3 日放量下跌（量比 >2×）→ 热度退潮信号

**③ 抛物线顶出信号（主动锁利）**
- 当日量比 >3× + 价格 >+30% + K 线上影线（收盘 < 日高 85%）→ Trim 50%
- 单日 gap +20% 以上（非基本面驱动）→ Trim 50% 锁利

> **动量止损原则**：热赛道热度来去快，技术信号为主，不要执着论文。

### 4. 止盈节奏（三类型完整版）

#### 🏦 蓝筹长持减仓节奏

```
持仓期间（正常持有，不轻易减仓）：
目标1到达（+10-15%）→ 轻减 25-33%（更多时候 Hold）
目标2到达（+30-50%）→ 视估值决定：
  - 估值仍合理 → Hold，等 LTCG 满 1 年
  - 估值已严重高估（P/FCF > 历史 1.5×）→ 减至 40-50% 剩余
LTCG 策略：持有满 1 年 → 应税账户节省 15-22% 税务成本
Stage 3 信号 + 基本面恶化 → 逐步减仓，不急于一次全出

止损后再入策略：蓝筹论文止损出后，若基本面修复（1-2 季度后验证）→ 可重新建仓。
```

#### 🚀 翻倍候选减仓节奏

```
目标1到达（+30-50%）→ 减仓 25-30%（只减一点，留大仓位跑）
目标2到达（+80-150%）→ 减仓至 40-50% 剩余，移动止损至成本价

【自由股规则 — 翻倍候选核心】
+100% 时 → 卖出 50%（锁利约等于初始本金）
剩余 50% 零成本持有（目标：3-5× 甚至更高）
零成本阶段止损：论文止损（成本为零，无需机械止损）
零成本阶段减仓：继续阶梯减（+200% Trim 25%，+300% Trim 25%，等）

Stage 3 信号出现 → 清仓（含自由股部分，因为赛道结构已变）
```

> **为什么翻倍候选只减 25-30%？** alpha 来自少数持仓跑 3-5 倍。T1 减太多 = 大幅削弱最终收益。

**【Chen Yun 补充风险纪律 — 翻倍候选专项】**

```
【止盈提醒 — 组合协同减仓（非单只止盈规则）】
触发条件：组合内 ≥2-3 只翻倍候选同时大幅上涨（各涨 >30%，合计浮盈显著）
动作：不依赖自由股规则被动等待，主动锁定部分本金 + 盈利
执行：分批卖出浮盈最大的 1-2 只（各减 20-30%），保留核心仓位继续跑
原则：高点主动降低风险敞口，不等回调再减（回调可能大幅吃掉浮盈）

【动态观望 — 宏观过滤层】
触发信号：VIX >35 / 地缘政治冲击 / SPY 跌破 200MA / 全球市场系统性动荡
动作：暂停新翻倍候选建仓；已持仓维持原止损，不强制减仓
持续期间：多看少动手，等待宏观信号改善
恢复信号：VIX 回落 <25 且 SPY 站回 200MA → 恢复正常操作节奏

【跳空理论 — 翻倍候选适用】
不追求跳空缺口补回 — 美股大幅跳涨后缺口常不回补
财报/催化剂跳涨后：
  论文仍成立 → Hold，不因"缺口应该补"而延迟持有
  估值已严重高估 → 主动 Trim，不等回调再减
  不能因为"等补缺"心理而放弃已到达的止盈目标
```

#### ⚡ 热赛道动量减仓节奏

```
目标1到达（+20-30%）→ 减仓 33%（动量短暂，快锁部分利润）
移动止损跟踪（热度窗口内）：
  价格创新高 → 止损上移，继续持有
  价格收盘跌破移动止损 → 剩余全出
赛道 ETF 跌破 MA50 → 全出，不恋战
抛物线顶信号 → Trim 50%，剩余出场移动止损跟踪
热度窗口结束（赛道失去 Chen Yun 关注或 ETF 趋势破坏）→ 全出
```

### 4.5 全局止盈信号（组合级别，优先于个股分析）

**来源：热门赛道方法论风险纪律**："多只核心持仓同时大幅上涨后，主动锁定本金 + 部分盈利，不追高。"

**触发条件（全局信号，独立于个股止损）：**

| 信号 | 阈值 | 建议行动 |
|------|------|---------|
| 多只持仓同时大涨 | 组合内 3+ 股票各涨 20%+ 且集中在 2-4 周内 | 全组合整体减仓 20-30%，锁定本金 |
| 赛道热度过高 | 同一赛道 ETF 单月涨幅 >30% | 赛道内持仓统一减 25-30%（轮动风险上升）|
| 宏观动荡/高不确定期 | 联储紧急声明 / 地缘升级 / VIX >35 | "多看少动手"——暂停新建仓，持仓保持止损纪律但不主动加仓 |

> **全局止盈的逻辑**：当多只仓位同时涨，通常是市场过热而非个股 alpha。主动锁利 = 保护已有收益，不是看跌。

---

**跳空理论（适用于 🚀 翻倍 / ⚡ 动量）：**

> 热门赛道方法论明确原则："忽略跳空补缺口逻辑。美股可以从缺口一路上涨，不需要等补缺口再行动。"

实操含义：
- **上涨跳空（gap-up）**：不要等"缺口回补"才买，破位上涨后直接按金字塔批次追入
- **下跌跳空（gap-down）**：不要因为"缺口总会补"而持有，论文破坏 + gap-down = 立即出
- **止损时**：价格跳空跌破止损位 → 以当日收盘价或开盘价出，不等回补

### 5. SEPA Stage 退出信号（按类型调整确认标准）

| 信号 | 🏦 蓝筹 | 🚀 翻倍 | ⚡ 动量 |
|------|--------|--------|--------|
| 跌破 150MA | 周线收盘 + 基本面恶化 | 连续 3 日跌破 | 单日收盘确认 |
| 死叉（50MA 穿 200MA）| 需同时有基本面恶化 | 死叉出现 | 死叉出现即评估全出 |
| 放量大跌（2x 均量）| 单日不构成信号，需看论文 | 连续 2 日 2× 放量下跌 | 单日 3× 放量下跌即评估 |
| Stage 4 确认 | + 基本面恶化才全出 | 进入 Stage 4 → 全出 | 进入 Stage 3 顶部 → 出 |

### 6. WebSearch 论文检查

搜索：`"[TICKER] earnings revenue guidance 2026"`

**按类型重点检查：**

蓝筹重点：
- 护城河有无新的竞争威胁？
- FCF 和利润率趋势是否恶化？
- 管理层是否有重大失误或战略转向？

翻倍重点：
- 核心催化剂是否仍然成立？
- 近期有无稀释融资（ATM / 定增）？
- 现金跑道是否缩短至危险区？
- 是否已触发自由股规则（+100%）？

动量重点：
- 赛道热度是否持续？（Chen Yun 近 7 日提及频率）
- 赛道 ETF 当前 vs MA50 状态？
- 是否有抛物线顶信号？
