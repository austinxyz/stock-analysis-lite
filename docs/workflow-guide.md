# 股票筛选、分析与买卖完整流程

四个 skill 构成完整的决策链：**发现 → 研究 → 入场 → 管理**。

---

## 全流程概览

```
/ticker-scan [赛道]          ← 从七大赛道中筛出候选
      ↓
/stock-analyze [TICKER]     ← 深度研究：基本面 + 技术面 + 翻倍路径
      ↓
/stock-entry [TICKER]       ← 计算入场区间、止损、建仓计划
      ↓
/stock-exit [TICKER]        ← 持仓管理：止盈/止损/减仓纪律
```

---

## 第一步：筛选候选 — `/ticker-scan`

**目的**：从七大结构性赛道中批量筛出翻倍/多倍候选，排除噪音。

### 用法

```
/ticker-scan 矿产资源          # 扫单一赛道
/ticker-scan                  # 扫全部七大赛道
/ticker-scan ABAT             # 对已知股票跑完整评分
/ticker-scan --tickers X Y Z  # 自定义候选池
```

### 三阶段漏斗

| 阶段 | 名称 | 通过标准 | 主要数据来源 |
|------|------|---------|------------|
| 2A | 九大翻倍股特征 | ≥2/9 特征命中 | yfinance + LLM 知识 |
| 2B | 六大类型 × 五大公式 | 类型 ≥3/6 AND 公式 ≥3/5 | yfinance + WebSearch backlog |
| 2C | 五大确定性标准 | ≥3✅ | yfinance + WebSearch |

### 输出标签

| 标签 | 含义 | 下一步 |
|------|------|--------|
| 🔥 多倍候选 | 2B 5+/6，公式 5/5，2C 4+✅ | 立即 `/stock-analyze` |
| ⭐ 翻倍候选 | 2B 3+/6，公式 3+/5，2C 3+✅ | 安排 `/stock-analyze` |
| 👀 观察 | 通过 Stage 2 但信号不足 | 等下季财报后重跑 |
| ❌ 筛除 | 未通过 2A 或 2B | 不跟踪 |

### 结果保存位置

```
outputs/ticker-scan/YYYY-MM-DD-[赛道].md
```

## 第二步：深度研究 — `/stock-analyze`

**目的**：判断市场是否对这只股票存在**错误定价**，以及翻倍路径是否清晰。

### 用法

```
/stock-analyze ABAT
/stock-analyze ABAT --type 翻倍   # 手动指定类型
```

### 三种股票类型（自动判断）

| 类型 | 判断条件 | 分析重点 |
|------|---------|---------|
| 🚀 翻倍候选 | 市值 <$10B + 热赛道 + 高成长/拐点 | 稀释、现金跑道、催化剂、Mauboussin BAIT |
| ⚡ 热赛道动量 | 市值 <$2B + 纯动量 + 热赛道卫星仓 | 赛道强度、SEPA Stage、短期催化剂 |
| 🏦 蓝筹长持 | 市值 >$10B + 已盈利 | 护城河、FCF 质量、估值 vs 历史 |

### Mauboussin BAIT 核心框架（翻倍候选重点）

找市场错在哪里：

| 维度 | 问题 | 强信号 |
|------|------|--------|
| **B — 行为错误** | 市场情绪过度悲观？锚定偏差？ | 股价从高点跌 60%+ 但业务未变 |
| **A — 分析盲点** | 分析师覆盖率低？指标被扭曲？ | <3名分析师，或 SBC 扭曲 EPS |
| **I — 信息不对称** | 10-Q 注脚隐藏信息？渠道数据？ | 非现金一次性项目被市场误读 |
| **T — 技术错位** | 指数剔除/强制清仓/期权到期？ | 短期技术性抛压，基本面未变 |

> B+I 双叠加最常见于翻倍股。A+T 更多见于蓝筹买入窗口。

### 翻倍候选专项检查清单

- [ ] **稀释风险**：近12月股份增加 X%？ATM 剩余额度？Warrant 潜在稀释？
- [ ] **现金跑道**：现金 $XM ÷ 月均 burn $XM = X 个月（<6月 🔴 危险）
- [ ] **催化剂日历**：未来12月内有哪些二元事件？各事件预计日期？
- [ ] **空头挤压潜力**：Float 小 + 空头占比高 = 额外上涨动力

### SEPA 技术面判断

| Stage | 技术状态 | 翻倍候选操作 |
|-------|---------|------------|
| Stage 1 | 底部横盘，均线趋平 | 关注区，等 Stage 2 启动信号 |
| Stage 2 | 均线多头排列，价格 > MA50 | 标准入场区，运行 `/stock-entry` |
| Stage 3 | 顶部走平 | 减仓信号 |
| Stage 4 | 下降趋势 | 回避 |

**Stage 1 等待条件**（需同时满足再进入下一步）：
1. 价格突破 MA200 并站稳 2-3 周
2. MA50 开始上穿 MA150
3. 催化剂或季报确认基本面改善

### 结果保存位置

```
wiki/tickers/[TICKER]/analysis.md    ← 常绿研究，可多次更新
```

## 第三步：计算入场 — `/stock-entry`

**目的**：给出精确入场区间、止损位、仓位大小和建仓计划。

### 用法

```
/stock-entry ABAT
```

需要先有 `wiki/tickers/[TICKER]/analysis.md`，否则先跑 `/stock-analyze`。

### ATR 自适应止损（核心改进）

不用固定百分比止损，用 ATR 适应波动率：

| 类型 | 止损宽度 | 原因 |
|------|---------|------|
| 🏦 蓝筹 | 轴心点下 3-5%（窄）| 护城河股护盘能力强 |
| 🚀 翻倍候选 | max(2×ATR, 10%)，上限 20% | 日波幅 5-10% 正常，固定止损会被洗出 |
| ⚡ 动量 | 近10日高 - 1.5×ATR（移动）| 追趋势，止损随涨上移 |

### 金字塔式建仓（翻倍候选三步）

```
批次1（试仓 40%）→ 轴心点突破时入，用小仓验证论点
        ↓
    价格上涨 8-12%（论点确认）
        ↓
批次2（确认 40%）→ 追入（"不怕追高，上涨是确认信号"）
        ↓
    催化剂事件确认（财报/合同/产品验证）
        ↓
批次3（加码 20%）→ 可选，论点已高度确认才加
```

> **无杠杆铁律**：三批合计不超过总资金风险上限，绝不借贷。

### 仓位计算（风险预算法）

```
翻倍候选：建议股数 = (总资金 × 1%) ÷ (入场价 - 止损价)
动量：    建议股数 = (总资金 × 0.5%) ÷ (入场价 - 止损价)
蓝筹：    建议股数 = (总资金 × 1.5%) ÷ (入场价 - 止损价)
```

### 结果保存位置

```
data/tickers/[TICKER]/entry-YYYY-MM-DD.md    ← 个人交易记录，gitignored
```

## 第四步：持仓管理 — `/stock-exit`

**目的**：在持有期间随时运行，评估止损/止盈/减仓状态。

### 用法

```
/stock-exit ABAT    # 检查当前持仓状态
```

### 三类型退出逻辑对比

| 项目 | 🏦 蓝筹 | 🚀 翻倍候选 | ⚡ 动量 |
|------|--------|-----------|--------|
| **止损触发** | 论文破坏（护城河受损）| 连续2日收盘 < 止损位 | 收盘 < 移动止损 |
| **MA 止损** | 周线 + 基本面恶化 | 连续3日跌破150MA | 单日收盘确认 |
| **目标1减仓** | +10-15%，减25-33% | +30-50%，减25-30% | +20-30%，减33% |
| **自由股规则** | 视估值 | **+100% 卖50%，剩余零成本** | 无（动量短期） |
| **持仓周期** | 1-5年 LTCG | 3-12个月 | 1-8周 |

### 自由股规则（翻倍候选核心）

```
+100% 时 → 卖出 50%（锁利≈初始本金）
剩余 50% → 零成本持有，目标 3-5×
零成本阶段 → 不设价格止损，只用论文止损
继续减仓 → +200% Trim 25%，+300% Trim 25%，...
```

### 全局止盈信号（组合级别）

当出现以下情况，**主动**减仓，不等个股止盈触发：

| 信号 | 阈值 | 行动 |
|------|------|------|
| 多只持仓同时大涨 | 3+ 股各涨 20%+ 集中在 2-4 周内 | 全组合整体减20-30% |
| 赛道过热 | 赛道 ETF 单月涨幅 >30% | 赛道内持仓统一减25-30% |
| 宏观动荡 | VIX >35 / SPY 跌破 200MA | 暂停新建仓，多看少动 |

### 跳空缺口原则

- **gap-down 跌破止损**：不等"缺口回补"，当日收盘价出
- **gap-up 论文不变**：不因"缺口要补"减仓，持有
- 热赛道方法论原则：美股常从缺口一路上涨，不回补

### 结果保存位置

```
data/tickers/[TICKER]/exit-YYYY-MM-DD.md    ← 个人交易记录，gitignored
```

## 实战场景示例

### 场景A：发现并建仓翻倍股

```
1. /ticker-scan 光互连
   → 结果：LWLG ⭐翻倍候选，2B 5/6，2C 4✅

2. /stock-analyze LWLG
   → BAIT：A+I双叠加（2名分析师，正毛利拐点未被定价）
   → Stage 1，等待条件：价格突破 MA200 $8.50 站稳

3. [等待：价格突破 MA200，MA50 开始上穿 MA150]

4. /stock-entry LWLG
   → 类型：🚀 翻倍候选，ATR%=7.2%
   → 止损：max(2×ATR=$1.20, 10%) = $1.20 → 止损 $7.30
   → 建仓计划：批次1 $8.80 × 45股，批次2 $9.60（+9%）× 45股，批次3 催化剂

5. 入场后，每次需要判断是否调整时：
   /stock-exit LWLG
   → 检查：论文是否完好，止损是否触发，自由股状态
```

### 场景B：持仓中评估是否减仓

```
场景：持有 ABAT，成本 $4.25，当前 $7.80（+84%），接近自由股规则

/stock-exit ABAT
→ 自由股状态：未触发（需 +100% = $8.50）
→ 止损：连续2日未跌破 $3.91 ✅
→ 论文：正毛利已维持3季，现金跑道12个月 ✅
→ 全局止盈：组合内另有2只涨20%+ ⚠️ 接近触发

建议行动：持有，等 $8.50 触发自由股规则（卖50%）
若全局止盈触发（第3只再涨20%），可提前减20%
```

## 常见问题

**Q：ticker-scan 找到 ⭐ 候选，什么情况下跳过 stock-analyze 直接 stock-entry？**
A：不建议跳过。ticker-scan 是量化粗筛，stock-analyze 才判断论点质量（稀释/现金跑道/BAIT）。没有 analyze，止损和目标价依据不足。

**Q：股票在 Stage 1，ticker-scan 给了 ⭐，现在要不要买？**
A：不建议。等 Stage 2 启动（价格突破 MA200 站稳 + 均线改善）再运行 stock-entry。Stage 1 的翻倍候选放入 watchlist，设置价格提醒。

**Q：已经买了，没有 entry 文件，怎么用 stock-exit？**
A：运行 `stock-exit [TICKER]`，它会提示你输入入场价、止损价、股票类型，不需要文件。

**Q：动量股和翻倍候选可以同时持有同一只吗？**
A：类型选一个。同只股票不同时用两套规则，否则止损逻辑冲突。通常 ticker-scan 赋予类型，stock-analyze 确认，entry 记录在案。
