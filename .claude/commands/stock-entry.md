# stock-entry

基于 SEPA + Moneyball 框架，给出具体入场区间、止损位、仓位建议。
**支持三种股票类型：🏦 蓝筹长持 / 🚀 翻倍候选 / ⚡ 热赛道动量**，对应完全不同的止损参数、减仓节奏和持仓心理框架。

## Arguments

Required: `<TICKER>` — 股票代码（如 `POET`, `NVDA`, `LLY`）
Optional: `--type 蓝筹 | 翻倍 | 动量` — 手动覆盖类型（否则从 analysis.md 读取）

## 框架依据

读取 `wiki/frameworks/sepa.md` 了解入场时机标准。
读取 `wiki/frameworks/moneyball.md` 了解非对称风险计算。

## Steps

### 1. 读取现有分析（如有）

检查 `wiki/tickers/[TICKER]/analysis.md` 是否存在，提取：
- **股票类型**（🏦 蓝筹长持 / 🚀 翻倍候选 / ⚡ 热赛道动量）— 关键，决定整个参数集
- 当前 Stage 和趋势模板得分
- 价格 vs 各 MA 数据
- 市值、ATR14、近期催化剂

如无此文件，先运行 `/stock-analyze [TICKER]`。

### 2. 拉取当前价格数据 + ATR（yfinance）

```python
import yfinance as yf
import pandas as pd

t = yf.Ticker("TICKER")
hist = t.history(period="1y")

# ATR-14（真实波幅均值）
high_low = hist['High'] - hist['Low']
high_close = abs(hist['High'] - hist['Close'].shift())
low_close  = abs(hist['Low']  - hist['Close'].shift())
tr = pd.concat([high_low, high_close, low_close], axis=1).max(axis=1)
atr14 = tr.rolling(14).mean().iloc[-1]
atr_pct = atr14 / hist['Close'].iloc[-1] * 100

price  = hist['Close'].iloc[-1]
ma50   = hist['Close'].rolling(50).mean().iloc[-1]
ma150  = hist['Close'].rolling(150).mean().iloc[-1]
ma200  = hist['Close'].rolling(200).mean().iloc[-1]
vol_avg20 = hist['Volume'].rolling(20).mean().iloc[-1]
hi52   = hist['High'].rolling(252).max().iloc[-1]
lo52   = hist['Low'].rolling(252).min().iloc[-1]

# 近10日最高（动量止损参考）
hi10d = hist['High'].rolling(10).max().iloc[-1]

market_cap = t.info.get('marketCap', 0)

print(f"价格={price:.2f} MA50={ma50:.2f} MA150={ma150:.2f} MA200={ma200:.2f}")
print(f"ATR14={atr14:.2f}  ATR%={atr_pct:.1f}%  近10日高={hi10d:.2f}")
print(f"52周高={hi52:.2f}  52周低={lo52:.2f}  市值={market_cap/1e9:.1f}B")
print(f"20日均量={vol_avg20/1e6:.1f}M")
```

### 3. 三类型参数对照（按 analysis.md 中的类型选择）

| 参数 | 🏦 蓝筹长持 | 🚀 翻倍候选 | ⚡ 热赛道动量 |
|------|-----------|-----------|------------|
| **入场条件** | SEPA ≥5/8，Stage 2 | SEPA ≥3/8，Stage 2（可接受 Stage 1 末期小仓） | Stage 2 且赛道 ETF >MA50，SEPA ≥3/8 |
| **止损宽度** | 轴心点下 3-5%（护城河支撑宽，不需宽止损）| max(2×ATR, 10%)，上限 20% | 移动止损：近10日高 - 1.5×ATR |
| **止损理念** | **论文止损优先**：业绩/竞争/护城河破坏 > 价格止损 | 2×ATR 止损 + 论文止损（谁先触发谁生效）| **移动止损**，技术破位（ETF 跌破 MA50）立即出 |
| **目标1（减仓）** | +10-15% 轻减（25-33%）| +30-50% 轻减（25-30%）| +20-30% 减仓 33%（动量快，早锁） |
| **目标2（剩余）** | +30-50% 视估值决定 | +80-150%（翻倍区）| 移动止损跟踪（热度窗口内尽量持有）|
| **翻倍后处理** | 视估值，LTCG 意识重调 | **自由股规则**：+100% 卖 50%，剩余零成本持有 | 无翻倍目标，热度退出优先 |
| **单笔风险** | 总资金 1.5%（大市值稳定，可略高）| 总资金 1%（缺口风险高）| 总资金 0.5%（热度退潮极快，小仓）|
| **持仓周期目标** | 1-5 年（LTCG 至少 1 年）| 3-12 个月 | 1-8 周（热赛道热度窗口）|
| **LTCG 提醒** | ✅ 强制意识：满 1 年再出，税差 15-22% | ⚠️ 关注：若持有 >1 年仍翻倍，LTCG 优先 | ❌ 短线为主，勿执着 LTCG |

### 4. SEPA 入场确认（参考 wiki/frameworks/sepa.md）

检查入场条件（按类型调整标准）：
- [ ] Stage 2 上升趋势（50MA > 150MA > 200MA，价格在 50MA 上方）
- [ ] 趋势模板得分达标（蓝筹/翻倍 ≥3/8，蓝筹 ≥5/8 更佳；动量 ≥3/8）
- [ ] 有明确轴心点（VCP / 杯柄 / 平台突破口）
- [ ] 成交量：整理期缩量，突破时放量（量比 ≥1.5×）
- [ ] **动量额外**：赛道 ETF 在 MA50 上方（赛道趋势确认）

**入场区间：** 轴心点上方 0–3% 为理想区。超出 1 ATR 以上不追。

### 5. Moneyball 风险计算（三类型版本）

```
【🏦 蓝筹长持】
止损位  = 轴心点 × (1 - 0.03~0.05)     ← 窄止损，护城河支撑
目标1   = 入场价 × 1.10~1.15             ← 轻减，不扰持仓
目标2   = 入场价 × 1.30~1.50             ← 视估值决定，LTCG 意识
R/R(T1) = (目标1 - 入场) / (入场 - 止损) → 要求 ≥2:1
LTCG 备注：计划持有 ≥1 年，出仓前确认税务账户（Roth IRA 最优）

【🚀 翻倍候选】
止损位  = 入场价 - max(2×ATR14, 入场价×0.10)，且不超过入场价 20%
目标1   = 入场价 × 1.30~1.50
目标2   = 入场价 × 1.80~2.50
R/R(T1) = (目标1 - 入场) / (入场 - 止损) → 要求 ≥2:1
自由股规则：达到 +100% 时 → 卖出 50% 锁利，剩余 50% 零成本持有
自由股备注：剩余无需设止损（成本归零），目标 3-5× 甚至更高

【⚡ 热赛道动量】
移动止损 = 近10日最高价 - 1.5×ATR14   （每日更新，价格创新高则上移）
初始止损 = 入场价 - 1.5×ATR14          （首日）
目标1   = 入场价 × 1.20~1.30             ← 快出部分（动量可能短暂）
目标2   = 移动止损跟踪                   ← 热度窗口内持有
抛物线顶出信号：单日量比 >3× + 价格 >T2 + 当日 K 线上影线 → Trim 50%
ETF 止损：赛道 ETF 跌破 MA50 → 不管个股价格，立即评估全出
```

### 6. 仓位建议（按类型）

```
【🏦 蓝筹长持】
单笔风险 = 总资金 × 1.5%
建议股数 = (总资金 × 1.5%) / (入场价 - 止损价)
说明：蓝筹波动低，可用略高风险预算，但总仓位不超过组合 25%

【🚀 翻倍候选】
单笔风险 = 总资金 × 1%
建议股数 = (总资金 × 1%) / (入场价 - 止损价)
说明：止损宽（2×ATR），缺口风险高；小仓保持弹药补加位

【⚡ 热赛道动量】
单笔风险 = 总资金 × 0.5%
建议股数 = (总资金 × 0.5%) / (入场价 - 初始止损价)
说明：热赛道波动极大，轻仓卫星；移动止损快速追踪，不需要多钱
```

### 6.5 金字塔式建仓（🚀 翻倍候选 / ⚡ 动量适用）

**来源：Chen Yun 方法论 + 热门赛道方法论（两框架建仓风格一致，互相印证）**：试仓 → 价格上涨确认 → 加仓 → 再次确认 → 加码。
**核心原则**："不怕追高"——上涨本身是论点确认信号，不是错过信号。
**Chen Yun 哲学基础**："先用小仓验证论点"——试仓不是为了便宜，而是用小代价测试论点是否成立；论点确认后才放大仓位，论点证伪则损失有限。
**与 SEPA 区别**：SEPA 要求精确轴心点突破；Chen Yun 方法论入场精度要求更低，更注重趋势持续性，允许在价格上涨后追入（追高是纪律）。

Section 6 计算的股数 = **总计划股数**（目标满仓），分批买入：

```
【🚀 翻倍候选 — 三步建仓】

批次1（试仓 / 论点验证）：总计划股数 × 40%
  → 时机：轴心点突破 + 量比≥1.5×（验证方向，不要求完美）
  → 目的：以小仓验证论点——若 2个交易日内收盘低于止损，论点证伪，损失有限；论点确认后才加码

批次2（确认加仓）：总计划股数 × 40%
  → 时机：批次1入场价上涨 8-12% 后追入（价格上涨 = 论点确认）
  → 条件：收盘价 > 批次1入场 + 1×ATR 且 MA50 仍支撑
  → "追高"是纪律，不是冲动

批次3（最终加码）：总计划股数 × 20%
  → 时机：催化剂事件确认（财报超预期、合同签署、产品验证）
  → 可选：若批次1+2已充分，批次3可跳过

⚠️ 止损位适用全部批次（止损不随批次2/3成本抬高而上移，除非触发移动止损逻辑）
⚠️ 批次2追入前必须确认：赛道 ETF 未跌破 MA50，论文无变化
⚠️ 无杠杆铁律：三批合计不得超过总资金风险上限

【⚡ 热赛道动量 — 两步建仓】

批次1（快入）：总计划股数 × 60%
  → 时机：动量初始突破 + 赛道 ETF 在 MA50 上方
批次2（追涨）：总计划股数 × 40%
  → 时机：价格创新高 + ETF 继续保持 MA50 上方
  → 若价格反转触及移动止损前，批次2可放弃

⚠️ 移动止损保护全部批次，一旦触发全部清仓

【🏦 蓝筹长持 — 两批（可选）】

批次1：总计划股数 × 60% → 轴心点附近入场
批次2：总计划股数 × 40% → 回调至 MA50 附近补仓（不追高）
→ 蓝筹不强制金字塔；价格持续上涨不回调则放弃批次2
```
