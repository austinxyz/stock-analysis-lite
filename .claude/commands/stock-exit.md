# stock-exit

基于 Moneyball 框架，给出翻倍候选股的止损触发逻辑、止盈节奏和减仓建议。

## Arguments

Required: `<TICKER>` — 股票代码（如 `ABAT`）

## Steps

### 1. 读取入场信息

检查 `data/tickers/[TICKER]/entry-*.md`（最新一份）提取：
- 入场价、止损位、目标1、目标2、入场日期
- 是否已触发自由股规则（+100%）

如无入场文件，让用户提供：入场价、止损价、当前持仓比例。

---

### 2. 拉取当前价格数据（yfinance）

```python
import yfinance as yf
import pandas as pd

t = yf.Ticker("TICKER")
hist = t.history(period="1y")

high_low   = hist['High'] - hist['Low']
high_close = abs(hist['High'] - hist['Close'].shift())
low_close  = abs(hist['Low']  - hist['Close'].shift())
tr = pd.concat([high_low, high_close, low_close], axis=1).max(axis=1)
atr14 = tr.rolling(14).mean().iloc[-1]

price  = hist['Close'].iloc[-1]
ma50   = hist['Close'].rolling(50).mean().iloc[-1]
ma150  = hist['Close'].rolling(150).mean().iloc[-1]
ma200  = hist['Close'].rolling(200).mean().iloc[-1]

vol_today = hist['Volume'].iloc[-1]
vol_avg20 = hist['Volume'].rolling(20).mean().iloc[-1]
vol_ratio = vol_today / vol_avg20

print(f"现价={price:.2f}  ATR14={atr14:.2f}({atr14/price*100:.1f}%)")
print(f"MA50={ma50:.2f}  MA150={ma150:.2f}  MA200={ma200:.2f}")
print(f"成交量比={vol_ratio:.1f}x")
```

---

### 3. 止损规则

**① 价格止损（噪音过滤版）**
- 连续 **2个交易日收盘** 低于止损位才确认触发（单日跌破不算）
- 止损位 = 入场时记录的 max(2×ATR, 10%) 位置

**② 论文止损（优先级更高，随时触发）**
- 核心催化剂消失（合同取消、竞争对手突破、管理层造假）→ 立即出
- 融资稀释严重（>20% 股份增发，且无对应业务价值）→ 立即出
- 现金跑道 <3 个月 → 评估是否出

**③ MA 止损**
- 连续 3 日跌破 MA150 → 出场

**跳空缺口原则：**
- **gap-down 跌破止损**：当日收盘价出，不等"缺口回补"
- **gap-up 论文不变**：持有，不因"缺口要补"减仓

**自由股阶段（+100% 后已卖 50%）：**
- 剩余 50% 零成本持有，不设价格止损
- 唯一出仓条件：论文彻底破坏，公司基本面不再支撑翻倍逻辑

---

### 4. 止盈节奏

```
目标1到达（+30-50%）→ 减仓 25-30%，留大仓位继续跑

目标2到达（+80-150%）→ 再减 25%，移动止损至成本价

【自由股规则 — 核心】
+100% 时 → 卖出 50%（锁利 ≈ 初始本金）
剩余 50% → 零成本持有，目标 3-5×
零成本后继续阶梯减：+200% Trim 25%，+300% Trim 25%...
```

> 为什么目标1只减 25-30%？翻倍股的 alpha 来自少数持仓跑 3-5 倍，T1 减太多会大幅削弱最终收益。

---

### 5. 全局止盈信号（组合级别）

不用等个股触发，出现以下情况主动减仓：

| 信号 | 阈值 | 行动 |
|------|------|------|
| 多只持仓同时大涨 | 3+ 股各涨 20%+ 集中在 2-4 周 | 全组合减 20-30% |
| 赛道过热 | 赛道 ETF 单月涨幅 >30% | 赛道内持仓统一减 25-30% |
| 宏观动荡 | VIX >35 / SPY 跌破 200MA | 暂停新建仓，多看少动 |

---

### 6. WebSearch 论文检查

搜索：`"[TICKER] earnings revenue guidance 2026"`

重点确认：
- 核心催化剂是否仍然成立？
- 近期有无稀释融资（ATM / 定增）？
- 现金跑道是否缩短至危险区？

### 7. 输出退出分析

保存到 `data/tickers/[TICKER]/exit-[YYYY-MM-DD].md`：

```markdown
# [TICKER] 持仓评估 — YYYY-MM-DD

## 持仓状态
- 入场价：$X（YYYY-MM-DD）| 持仓天数：X 天
- 当前价：$X（+X% / -X%）
- 自由股状态：[未触发 / ✅ 已触发，50% 已出，50% 零成本持有]

## 止损检查
- [ ] 连续2日收盘低于止损位 $X：[是/否]（今日=$X）
- [ ] 论文破坏（催化剂消失/稀释严重/跑道危险）：[是/否]
- [ ] 连续3日跌破 MA150 $X：[是/否]
- ATR14：$X → 单日正常波动 ±$X

## 止盈进度
| 目标 | 价位 | 状态 |
|------|------|------|
| 目标1（减 25-30%）| $X（+30-50%）| [已到达 / 未到达，距 X%] |
| 目标2 | $X（+80-150%）| [状态] |
| 自由股触发 | $X（+100%）| [未触发 / ✅ 已触发] |

## Stage 信号
- 当前 Stage：[2保持 / 3警告 / 4清仓]
- MA50=$X | MA150=$X | MA200=$X

## 全局止盈检查
- 组合内同时涨 20%+ 的持仓数：X 只 → [未触发 / ⚠️ 考虑整体减仓]
- 赛道 ETF 单月涨幅：X%（>30% = 赛道过热）
- 宏观状态：[正常 / ⚠️ VIX>35]
- 跳空情况：[无 / gap-down 跌破止损 → 不等补缺口，当日收盘出]

## 建议行动
[持有 / 减仓至 X% / 触发自由股规则（卖50%）/ 全部清仓] — 理由：[1句]

> ⚠️ 单日触及止损位不等于触发，连续2收盘低于才确认。
> ⚠️ 论文完好时，价格止损是最后一道线，不是第一道线。
> ⚠️ gap-down 跌破止损不等补缺口，以当日收盘价出。
```
