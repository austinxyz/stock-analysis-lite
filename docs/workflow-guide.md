# 用 AI 寻找翻倍股：四步流程指南

> **免责声明：** 本指南结合炒股理论与 AI 分析，提升研究效率，仅作辅助参考，不构成具体投资操作建议。投资有风险，决策需自主判断。

---

翻倍股的核心逻辑：**市场对某只股票存在错误定价** → 发现它 → 研究确认 → 在合适时机买入 → 纪律性管理持仓。

四个 skill 对应四步：

```
/ticker-scan [赛道]      ← 从热门赛道批量筛出候选
      ↓
/stock-analyze TICKER   ← 研究：市场错在哪里？翻倍路径清晰吗？
      ↓
/stock-entry TICKER     ← 入场：买哪里？止损在哪？买多少？
      ↓
/stock-exit TICKER      ← 管理：什么时候减仓？什么时候止损？
```

---

## 第一步：找候选 — `/ticker-scan`

从七大结构性赛道（AI芯片、光互连、矿产资源、生物医药等）中批量筛选翻倍候选，用三阶段漏斗过滤噪音。

**用法示例：**
```
/ticker-scan 光互连        # 扫单一赛道
/ticker-scan               # 扫全部七大赛道
/ticker-scan ABAT          # 对已知股票评分
```

**三阶段漏斗：**

| 阶段 | 内容 | 通过标准 |
|------|------|---------|
| 2A | 九大翻倍股特征 | ≥2/9 命中 |
| 2B | 六大类型 × 五大公式 | 类型 ≥3/6 且公式 ≥3/5 |
| 2C | 五大确定性标准 | ≥3 项✅ |

**结果标签：**

| 标签 | 含义 | 下一步 |
|------|------|--------|
| 🔥 多倍候选 | 2B 5+/6，公式 5/5，2C 4+✅ | 立即 `/stock-analyze` |
| ⭐ 翻倍候选 | 2B 3+/6，公式 3+/5，2C 3+✅ | 安排 `/stock-analyze` |
| 👀 观察 | 信号不足 | 等下季财报后重跑 |
| ❌ 筛除 | 未通过 2A 或 2B | 不跟踪 |

结果保存：`outputs/ticker-scan/YYYY-MM-DD-[赛道].md`

## 第二步：研究确认 — `/stock-analyze`

核心问题：**市场错在哪里？** 和 **翻倍路径是什么？**

**用法：**
```
/stock-analyze ABAT
```

### BAIT 框架 — 找市场的错误

| 维度 | 意思 | 强信号例子 |
|------|------|-----------|
| **B — 行为错误** | 市场情绪过度悲观 | 股价从高点跌 60%+ 但业务未变 |
| **A — 分析盲点** | 分析师覆盖少/指标被扭曲 | <3名分析师；SBC 非现金亏损被误读 |
| **I — 信息不对称** | 10-Q 注脚有隐藏信息 | 一次性非现金项目让净亏损看起来很大 |
| **T — 技术错位** | 技术性抛压，基本面没变 | 指数剔除/强制清仓导致超卖 |

> 翻倍股最常见：**A + I 双叠加**（覆盖率低 + 财务数字被误读）

### 翻倍股专项检查

- **稀释风险**：近12月股份增加多少？ATM 剩余额度？Warrant 潜在稀释？
- **现金跑道**：现金 ÷ 月均 burn = X 个月（< 6个月 🔴 危险）
- **催化剂日历**：未来12月有哪些关键事件？（财报、合同、FDA 等）
- **空头挤压潜力**：Float 小 + 空头比例高 = 额外上涨动力

### 技术面：等 Stage 2 再入

| Stage | 状态 | 操作 |
|-------|------|------|
| Stage 1 | 底部横盘，均线趋平 | 加入 watchlist，等待 |
| **Stage 2** | **均线多头排列，价格 > MA50** | **运行 `/stock-entry`** |
| Stage 3 | 顶部走平 | 减仓信号 |
| Stage 4 | 下降趋势 | 回避 |

Stage 2 启动信号（需同时满足）：
1. 价格突破 MA200 并站稳 2-3 周
2. MA50 开始上穿 MA150
3. 有催化剂或财报确认基本面改善

### Moneyball 情景 EV — 翻倍路径的概率化

| 情景 | 目标价 | 概率 |
|------|--------|------|
| 🐂 Bull | $X | 20-35% |
| Base | $Y | 45-60% |
| 🐻 Bear | $Z | 15-30% |

**EV = Σ(目标价 × 概率)**。买入触发线：**现价 ≤ EV × 0.85**（15% 安全边际）才可执行入场；高于此线即使技术面好也不追。

研究报告保存：`wiki/tickers/[TICKER]/analysis.md`

## 第三步：入场计划 — `/stock-entry`

给出精确的入场区间、止损位、建仓节奏。

**用法：**
```
/stock-entry ABAT
```

### 止损：用 ATR 适应波动，不用固定百分比

翻倍股日波幅 5-10% 很正常，固定 5-8% 止损很容易被洗出。

```
止损宽度 = max(2 × ATR14, 10%)，上限 20%
```

ATR14 = 过去14天平均真实波幅，反映该股"正常日波动"。

### 入场前三道闸（顺序检查）

1. **财报距离**：财报前 2 周不入场（`earnings_in_days ≤ 14` → 等财报后）
2. **市场环境**：SPY+QQQ 都在 MA200 上 = bull（单笔风险 1%）；都在下 = bear（不建新仓）；其余 chop（0.5%）
3. **PW EV 触发线**：现价 > EV × 0.85 → 不追

### 金字塔式建仓 — 三步进场

```
批次1（40%）→ 轴心点突破 + 量比 ≥1.5× 时入场（试仓）
      ↓ 价格上涨 8-12%（论点确认）
批次2（40%）→ 追入（上涨是确认信号，不怕追高）
      ↓ 催化剂事件确认（财报/合同/产品）
批次3（20%）→ 可选，论点高度确认才加
```

> **无杠杆铁律**：三批合计不超过总资金风险上限，绝不借贷。

### 仓位大小（风险预算法）

```
建议股数 = (总资金 × 1%) ÷ (入场价 - 止损价)
```

例：总资金 $50,000，入场价 $10，止损 $8.50（止损 $1.50）
→ 建议股数 = ($50,000 × 1%) ÷ $1.50 = **333 股**

## 第四步：持仓管理 — `/stock-exit`

持有期间随时运行，评估是否该减仓或止损。

**用法：**
```
/stock-exit ABAT
```

### 止损规则

- **价格止损**：连续 **2日** 收盘价低于止损位 → 出场（单日不算，防噪音）
- **MA 止损**：连续 3日 跌破 MA150 → 出场
- **跳空下跌**：gap-down 且触发止损 → 当日收盘价出，**不等缺口回补**

### 止盈节奏

| 阶段 | 触发条件 | 动作 |
|------|---------|------|
| 减仓1 | +30-50% | 卖出 25-30%，锁定部分利润 |
| 减仓2 | +80-100% | 卖出 25%，进入"自由股"状态 |
| **自由股规则** | **+100%（翻倍）** | **卖出 50%，剩余零成本持有** |
| 继续持有 | 零成本后 | 目标 3-5×，只用论文止损 |

**自由股规则详解：**
```
成本 $5 → 涨到 $10（+100%）→ 卖出一半（回收本金）
剩余一半 → 零成本，不再设价格止损
→ 目标 3× ($15)、5× ($25)，让利润奔跑
```

### 全局止盈信号（组合级别）

不用等个股止盈触发，以下情况主动减仓：

| 信号 | 阈值 | 行动 |
|------|------|------|
| 多只持仓同时大涨 | 3+ 股各涨 20%+ 集中在 2-4 周 | 全组合减 20-30% |
| 赛道过热 | 赛道 ETF 单月涨 >30% | 赛道内持仓减 25-30% |
| 宏观动荡 | VIX >35 | 暂停新建仓，多看少动 |

## 完整流程示例

**场景：从零开始发现并操作 ABAT**

```
1. /ticker-scan 矿产资源
   → ABAT ⭐翻倍候选，2B 4/6，2C 3✅

2. /stock-analyze ABAT
   → BAIT：A+I双叠加
     - A：仅2名分析师覆盖，正毛利拐点未被定价
     - I：Q3 净亏损 $33.8M 中 $24.5M 是一次性非现金 SBC，
          剔除后真实亏损约 $9.9M
   → 技术：Stage 1（均线空头排列），等待 Stage 2
   → 等待条件：价格突破 MA200 $3.88 站稳

3. [等待 2-3 周：价格突破 MA200，MA50 开始改善]

4. /stock-entry ABAT
   → ATR14 = $0.28，止损 = max(2×$0.28, 10%) = 10%
   → 入场 $4.20，止损 $3.78
   → 批次1：$4.20 × 200股（试仓）
   → 批次2：$4.60（+9.5%）× 200股（追入）
   → 批次3：财报确认后 × 100股（可选）

5. 持仓管理（随时运行）：
   /stock-exit ABAT
   → 当前 $6.30（+50%）→ 减仓25%，锁定部分利润
   → 等 $8.40（+100%）→ 触发自由股规则，卖50%
```

## 常见问题

**Q：ticker-scan 找到候选，可以跳过 stock-analyze 直接入场吗？**
A：不建议。ticker-scan 是粗筛，stock-analyze 才验证"市场的错误"在哪里。没有这一步，止损位和目标价都缺乏依据。

**Q：Stage 1 的候选要不要提前建仓？**
A：不建议。Stage 1 还在底部，上涨信号未确认，等 Stage 2 启动再进。先加 watchlist 设提醒。

**Q：已经持仓，涨了很多，不想卖但又怕跌，怎么办？**
A：运行 `/stock-exit`，它会根据当前价格和止损位给出具体建议。+100% 就用自由股规则——卖一半回本，剩下的零成本持有，心理压力自然消失。

**Q：gap-down 大跌，要等回补缺口再出吗？**
A：不等。触发止损就当日收盘出，缺口不一定回补（美股热赛道股常从缺口一路继续跌）。
